# Unbeatable-Tic-Tac-Toe
***
An unbeatable, single-player Tic-Tac-Toe game.
The AI algorithm driving the game is the Minimax Algorithm.

* [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
